Documentación API

La API de Pullso te permite ejecutar análisis de reportes, listar resultados y descargar PDFs de forma programática.

Autenticación por API key: envía el header X-API-Key: tu_key o Authorization: Bearer tu_key en cada petición. La clave se obtiene tras solicitar acceso; un administrador la asigna desde el panel Admin → Acceso API. Uso limitado por estándares de industria (p. ej. 60 req/min, 1000/día).

Endpoints principales: POST /api/v1/analyze (subir reportes y obtener análisis), GET /api/v1/analyses (listar), GET /api/v1/analyses/{id} (detalle), GET /api/v1/analyses/{id}/pdf (descargar PDF).
