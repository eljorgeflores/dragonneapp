# Assets informe revenue (Dragonné)

Coloca aquí los PNG oficiales para no mezclar logos claros/oscuras:

| Archivo | Uso |
|---------|-----|
| `logo-for-dark-bg.png` | Wordmark **claro** sobre portada oscura (PDF/HTML). |
| `logo-for-light-bg.png` | Wordmark **oscuro** para cabeceras en páginas claras (PDF). |
| `isotipo.png` | (Opcional) isotipo para futuras variantes. |

Si faltan, el sistema usa `static/branding/dragonne-wordmark.png` como respaldo.

Tras cambiar archivos, no hace falta tocar código: rutas en `services/revenue_report/brand_assets.py`.
